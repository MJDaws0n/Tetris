# Tetris

This is a simple Tetris game built with Neutralino.js.

## How to Run

1. **Open a terminal and navigate to the `tetris` folder.**

2. **Choose the Neutralino binary for your platform from the `bin` folder:**

   - For macOS (Apple Silicon): `neutralino-mac_arm64`
   - For macOS (Intel): `neutralino-mac_x64`
   - For macOS (Universal): `neutralino-mac_universal`
   - For Linux (x64): `neutralino-linux_x64`
   - For Linux (ARM64): `neutralino-linux_arm64`
   - For Linux (ARMHF): `neutralino-linux_armhf`

3. **Move the correct binary into the `tetris` folder (not inside `bin`):**

   For example, on macOS (Apple Silicon):
   ```sh
   mv bin/neutralino-mac_arm64 .
   ```
   Or for your platform, replace the filename as needed.

4. **Make sure the binary is executable:**
   ```sh
   chmod +x neutralino-*
   ```

5. **Run the binary:**
   ```sh
   ./neutralino-mac_arm64
   ```
   (or the correct filename for your platform)

6. **The game window should open automatically.**

---

Enjoy the game!
